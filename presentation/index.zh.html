客户端技术

Yeoman, Gulp, Bower, AngularJS, Bootstrap

Yeoman

Yeoman 做为应用的 生成工具

  • 数百个生成工具可以利用
  • 主要面向 JavaScript 前端应用程序
  • 好评率高的生成工具质量可靠
                            yo jhipster

Gulp

Gulp 是一个 JavaScript 任务管理工具

  • 开发环境下, 它可以让你实时的编辑代码，同时在后台运行测试用例...
  • 生产环境下, 它可以压缩和优化所有的资源文件，编译你的 Sass 代码等...
                            gulp test

Bower

Bower 是一个 JavaScript 包管理工具

  • 你可以搜索，安装和更新用到的 JavaScript 库
  • 它提供了 大量 的第三方库
                            bower install angular

Karma

Karma 用于运行 JavaScript 的单元测试

  • 它可以运行于 Phantom.js , 一个 headless 的浏览器
  • 它 非常的 快, 而且可以持续的在后台运行
                            karma start src/test/javascript/karma.conf.js

AngularJS

最流行的 JavaScript 框架

  • 强大 & 容易学习
  • 数据绑定, 表单验证, i18n... 这些都是开箱即用的

服务器端技术

Maven, Spring, Spring MVC REST, Spring Data JPA, Netflix OSS

Maven | Gradle

最流行的Java构建工具

  • 预定义的代码目录和构建目标
  • 集成了主流的 IDEs
  • 强大的插件系统, 包括 Spring Boot 和 Yeoman
                            
./mvnw spring-boot:run
                            
                            
./gradlew bootRun

Spring Boot

Spring 是 java 应用程序事实上的标准

  • IoC, AOP 和抽象层简化应用编程
  • 快速, 轻量, 需要的时候 也可以变的很强大
  • Spring Boot 提供开箱即用的配置
  • 应用程序热加载功能
                            
@Service
@Transactional
public class UserService {

    @Inject
    private UserRepository userRepository;

}

Netflix OSS

Netflix OSS 提供开发微服务的工具和框架

  • Eureka - 服务注册功能可用于实现负载均衡和故障恢复
  • Zuul - 代理功能实现动态路由，监控和安全
  • Ribbon - 用软件实现服务的负载均衡

Liquibase

数据库更新变的更加容易

  • 处理数据库 schema 的更新
  • 很好的支持 Spring 和 JPA
  • 需要对 数据库和 ORM 有良好的理解

JPA

Java 技术栈标准的 ORM 解决方案

  • 默认支持 Hibernate
  • 非常完整，令人印象深刻的一套功能
  • 对于新手理解起来可能还是有点复杂
                            
@Entity
public class User implements Serializable {

    @Id
    private String login;

}

Spring Data JPA

在 JPA 的基础上提供了更多的语法糖

  • 自动为你生成 JPA repositories 代码
  • 移除了大量的样板代码
                            
public interface PersistenceAuditEventRepository extends JpaRepository<PersistentAuditEvent, String> {

    List<PersistentAuditEvent>
        findByPrincipalAndAuditEventDateGreaterThan(String principal, LocalDateTime after);

}

NoSQL 提供可扩展性

  • 更具可扩展性，更容易使用

  • SQL 数据库和 JPA 之外的第二种选择

  • Ehcache 是最广泛使用的解决方案
  • HazelCast 是另一个不错的选择，支持集群

让我们来创建第一个应用

10 分钟教程

安装 Yeoman

                            
yarn global add yo
yarn global add generator-jhipster

创建一个应用

并用嵌入的 Maven wrapper 来运行
(自动支持应用的热加载) 

                            
yo jhipster
./mvnw

客户端热加载

用 Webpack 和 Browsersync 实现客户端代码(HTML, CSS, JavaScript)的热加载。 编辑你的代码，浏览器会自动刷新: 

                            
yarn start

添加一个实体

                            
yo jhipster:entity Foo

生产环境中运行

(用 Docker 来运行数据库)

                            
./mvnw -Pprod package
docker-compose -f src/main/docker/mysql.yml up -d
java -jar target/*.war --spring.profiles.active=prod
